1. Introduction

1-1 Explanation of .NET 7

Microsoft has released the production version of .NET 7, the latest version of its cross-platform, open-source software development platform. This update brings a number of enhancements including performance improvements, C# language upgrades, and cloud-native, container-based development capabilities. The key themes of this release include enabling developers to write performant web APIs, building cloud-native applications, and deploying them across Linux. Additionally, developer productivity and containerization of projects are emphasized, along with setting up CI/CD workflows in GitHub actions. Available for Windows, Linux, and macOS, .NET 7 is the third major release in Microsoftâ€™s .NET unification journey, which allows developers to learn once and reuse skills with one SDK, one runtime, and one set of base libraries to build all kinds of applications including cloud, web, desktop, mobile, artificial intelligence, and IoT.
